## För att starta appen
Steg 1: npm install
Steg 2: npm run dev


### För G
- Uppfylla alla krav på funktionalitet Gjort i Typescript och i sass, Använda dig av statiska typer och interface, använda dig av scss variabler, nesting, och mixin, modules
- API-svaret (arrayen du får tillbaka) ska ha en datatyp och får inte vara any. Ex: const data: Dog[] = await response.json();

### För VG
- Ha delat upp din kod i moduler i Typescript, alla interface ska ligga i en egen modul och importeras.
- alla sass filer skall ha sin egen modul
- kombinera interface och använda dig av generics
- Skall använda sig av någon typ av bibliotek som tex Axios för att hämta data.
